## 🤔 Seedance 2.0 とは？

**Seedance 2.0** は**ByteDance**が開発した動画生成モデルで、業界初の**4モーダル同時入力**をサポートするモデルです — 画像、動画、音声、テキスト。

**Key Features:**
- 🎥 **テキストから動画** — テキスト説明から動画を生成
- 🖼️ **画像から動画** — 静止画をダイナミックな動画にアニメーション化
- 📹 **動画から動画** — 既存の動画を変換・拡張
- 🎵 **音声駆動** — 音声入力で動画を生成
- 📐 **最大1080p解像度**、4〜15秒の長さ
- 🔊 **自動吹き替え＆スコアリング** — 自動ナレーションとBGM

### No. 1: Seedance 2.0：15秒で描く、映画のような日本のロマンス短編映画

![中文](https://img.shields.io/badge/lang-中文-red)
![Featured](https://img.shields.io/badge/⭐-Featured-gold)

#### 📖 説明

Seedance 2.0 向けの、非常に詳細な 15 秒間のマルチシーンプロンプト。映画のような超リアルな日本の高校生の純愛短編映画を生成するように設計されています。プロンプトは、シーン設定（誰もいない教室、暖かく金色に輝く太陽の光、舞い上がる埃）、カメラの動き、キャラクターの一貫性（変形やずれがないこと）、繊細な微表情、同期した呼吸と唇の動き、セリフ、効果音（蝉の声、ペンが紙を擦る音、低周波の心臓の鼓動、柔らかなピアノの音）を指定しています。物語は、書いている少女と、彼女を密かに見つめる少年との間の、強烈でぎこちなく、親密な感情的緊張に焦点を当て、最終的には内気な対決へと発展します。

#### 📝 プロンプト

```
15秒の映画のような日本のドラマ、純愛の曖昧な短編映画、超リアルな画質、午後の人気のない教室に差し込む温かい金色の太陽の光がブラインドを通して並んだ机に降り注ぎ、光の筋の中を細かな埃がゆっくりと舞い上がる。古い木製の机、極めて自然で繊細な動き、呼吸、目の緊張感。登場人物の顔、服装、髪型は変形、ずれ、アーティファクトなく一貫して維持され、呼吸に合わせて胸がわずかに上下する様子もリアル。浅い被写界深度、クリーミーにぼやけた背景、温かいフィルムグレイン、8Kのシャープさ、日本の青春の抑えられた胸キュンするような息苦しい雰囲気。
0-4秒：デスクトップのミディアムショットから、並んで座る二人の横顔のクローズアップへと、極めてゆっくりとプッシュインショット。夏服の制服を着た純粋な少女がうつむいてノートに集中して書き込んでおり、長い黒髪と耳元の後れ毛が微風に優しくなびく。長いまつげが繊細な影を落とし、肌は自然なピンク色で柔らかく、集中しているために無意識に口角がわずかに上がっている。呼吸は軽く均一。
4-9秒：少年のクローズアップに切り替わる。制服の襟はわずかに緩んでおり、彼は机に肘をついて、こっそりと顔を向けて彼女を見つめている。彼の瞳には優しく抑えられた愛情と優しさが満ちており、瞳孔はわずかに開いている。喉仏が優しく上下する。突然、彼女のペンが止まったことに気づき、慌てて自分のノートを見るふりをして顔をそらす。耳たぶはすぐにわずかに赤くなり、ペンを握る指先はわずかに震えている。前髪の間から時折彼女をちらりと見やり、呼吸はわずかに乱れ、落ち着こうと唇をきつく閉じている。
9-15秒：同じフレームに二人の顔の超クローズアップ、スローモーションで突然目が合う：少女はゆっくりと顔を向け、最初はぼんやりとした驚きの表情を見せるが、すぐに恥ずかしそうに0.3秒間うつむき、下唇を優しく噛む。頬と耳たぶは瞬時に桜色に染まり、潤んだまつげがはにかむように再び彼を見上げ、柔らかく恥ずかしそうに「…何見てるの？」とささやく。少年は完全に固まり、瞳孔が開き、0.4秒間呆然とした後、慌てて静かにどもりながら「な、なんでもない…」と答える。少女はさらに小さな声で、唇を噛みながら再び彼をちらりと見て、ささやき続ける。「…嘘つき。」少年は一瞬の間を置き、それから優しくため息をつき、ささやく。「…君を見てただけだよ。」彼の口角はゆっくりと上がり、はにかむような、優しい、少し歪んだ笑顔を見せる。目尻には細い線が現れ、呼吸は明らかに深くなる。見えない電流が二人の顔の間の曖昧な緊張を引き寄せ、互いの呼吸の温度を分かち合う。背景はクリーミーで夢のような光の斑点、温かいハロー、そして微細な空気の粒子が幾重にも重なって完全に溶け込んでいる。
リップシンクは自然で正確、感情的な微細な震えと呼吸は同期しており、セリフは恥ずかしそうなトーンの低エネルギーなささやき声で、200-400ミリ秒の自然な短い間があり、話すときに口はわずかに動くだけで、誇張やロボットのような不自然さはなく、完璧な自然なリップシンクと感情の信憑性。
全体的な効果音：遠くでかすかに聞こえる夏のセミの鳴き声、ペンが紙に触れる柔らかい擦れる音、ほとんど聞こえない心臓の低周波の鼓動、最後に非常に軽やかで風通しの良いピアノの音にフェードアウトする。セリフはささやき声として完全に自然にシーンに溶け込んでおり、少女の声は柔らかく恥ずかしがり屋で、少年の声は慌てたどもりから優しい声へと変化する。
キャラクターのアイデンティティは一貫して維持され、リアルで繊細な頭の傾き、目の動き、呼吸の同期があり、テキスト、透かし、字幕はなく、純粋な日本スタイルの青春の秘めた恋の胸キュンサスペンス。
```

### No. 2: ハリウッド オートクチュール ファンタジー ビデオ プロンプト

![中文](https://img.shields.io/badge/lang-中文-red)
![Featured](https://img.shields.io/badge/⭐-Featured-gold)

#### 📖 説明

Seedance 2.0 向けの、ハリウッドのオートクチュールファンタジー映画を制作するためにデザインされた、詳細かつ複数シーンにわたる動画生成プロンプト。このプロンプトは、スタイル、解像度 (8K)、レンダリングエンジン (Unreal Engine 5)、再生時間 (15 秒) を指定し、液体状の青と白の磁器をまとったモデルが、飛び散って水墨画のツバメになり、最終的に 3D の流体インクの渦を形成するという、3 つの異なるカメラ/アクションシーケンスを含みます。

#### 📝 プロンプト

```
[スタイル] ハリウッドのオートクチュール ファンタジー超大作、8K 超高精細、フォトリアリスティック、ハイファッション エディトリアル スタイル、Unreal Engine 5 の流体レンダリング、視覚的錯覚。[尺] 15 秒。[シーン] 果てしなく広がる、現実のウユニ塩湖（スカイミラー）の塩原。空には重苦しい暗い雲が立ち込め、地面はすべてを鏡のように完璧に映し出し、全体的にミニマリストでクールなトーンの絵が広がる。[00:00-00:05] ショット 1: オートクチュールの登場と陶器のような肌。カメラ位置: 極端なローアングルからの上向きショット、超望遠レンズでズームイン。アクション: 非常に特徴的なハイファッションの顔を持つアジア人女性モデルが、水面をクールに歩く。効果: 彼女は布ではなく、流れるような本物の液体青白磁でできたロングドレスを身につけている。歩くたびに、スカートは本物の陶器のような澄んだ衝突音を立て、表面には流れるような光沢がある。伝統的な青と白の模様が、白い陶器のような質感のスカートの上を生きているかのように動き回る。[00:05-00:10] ショット 2: 物理的な粉砕と水墨画のような降下。カメラ位置: 顔の極端なクローズアップ、フォーカスが急速に引き戻される。アクション: モデルが突然立ち止まり、冷たくカメラを見つめ、指をパチンと鳴らす。効果: 指を鳴らした瞬間、彼女の青白磁のドレスは落ちることなく、瞬時に何千もの極めてフォトリアリスティックな水墨画のツバメに爆発する。これらのツバメは本物の水滴と墨の跡を運び、空気中に黒い流体の残像を引きずりながら、彼女の周りを狂ったように旋回する。[00:10-00:15] ショット 3: 次元的な溶解と深淵の反射。カメラ位置: 高高度からのオーバーヘッドショット、カメラが急速に回転し下降する。アクション: 水墨画のツバメの群れが、モデルの足元の鏡のような湖水に飛び込む。効果: 元々固かった塩湖の表面張力が瞬時に消滅する。極めてリアルな世界全体が、澄んだ水に濃い墨を落としたように激しく滲み、溶解し始める。現実の暗い雲とモデルの姿は、極めて壮大な 3D 流体墨渦に完全に変容し、カメラを白と黒が織りなす深淵へと完全に飲み込む。
```

### No. 3: 現代の田舎の美学が癒しをもたらす短編映像プロンプト

![中文](https://img.shields.io/badge/lang-中文-red)
![Featured](https://img.shields.io/badge/⭐-Featured-gold)

#### 📖 説明

Seedance 2.0 用の、現代的な田園美学スタイルで癒しと映画のような短編フィルムを生成するための、詳細な 3 ショットプロンプトです。スタイル（シネマティックコマーシャル、4K/8K、エクストリームマクロ、自然光、ASMR）、設定（庭の見えるモダンなオープンキッチン）、登場人物（リネンを身につけた集中力のあるクリエイター）、そして 3 つのシーンでの具体的なアクション（トマトを摘む、正確にカットする、静かに食べる瞬間）が指定されています。

#### 📝 プロンプト

```
[スタイル]
モダンな田舎の美学、映画のようなコマーシャル品質、Sony A7S3/シネマカメラで撮影、4K/8K の超クリアな画質、エクストリームマクロ、自然で透明感のあるライティング、癒しの ASMR、時代劇のような雰囲気はなし。

[シーン]
手入れの行き届いたモダンな農家のオープンキッチン、背景には青々とした菜園、明るい日差し。

[登場人物]
モダンな田舎のクリエイター、黒い長い髪を木製のヘアピンで無造作にまとめ、ダークブルーの快適なリネン服を着用、クリアなメイク、集中した穏やかな瞳。

[00:00-00:05] ショット 1：朝の収穫（新鮮さ）
映像：高精細なクローズアップ。朝の太陽が植物にサイドバックライトを当てる。

詳細：非常にシャープなフォーカスで、トマトの表面の産毛と水滴が滑り落ちる軌跡がはっきりと見える。背景はぼかされた高品質の緑。

詳細：マクロレンズが、包丁の刃が食材を切り裂き、汁が飛び散る瞬間を捉える。その後、土鍋のストーブでオレンジ色の炎が揺らめく様子に切り替わり、光と影は暖かくリアル。

雰囲気：逆光の中で湯気がゆっくりと立ち上り、風の音さえ聞こえそうなほど静かなシーンは、現代人が切望する究極のリラックス感を表現している。
```
