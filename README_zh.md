## 🤔 什么是 Seedance 2.0？

**Seedance 2.0** 是**字节跳动**开发的视频生成模型，是业内首个支持**四模态同时输入**的模型——图片、视频、音频和文本。

**Key Features:**
- 🎥 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片动画化为动态视频
- 📹 **视频转视频** — 转换和扩展现有视频
- 🎵 **音频驱动** — 通过音频输入驱动视频生成
- 📐 **最高 1080p 分辨率**，4-15 秒时长
- 🔊 **自动配音配乐** — 自动旁白和背景音乐

```
请生成一张高完成度、高清的“街头花式足球 16 步动作拆解”海报风格分镜。布局应为 4x4 的十六宫格排列，每个格子展示同一位街头花式足球表演者在同一场景下的连续动作步骤。整张图片应看起来像“高端运动海报 + 技术拆解教学项目”的结合体，既具有视觉冲击力，又具备清晰的教学动作说明。

[整体设定]
主题为：街头花式足球表演，非正式比赛，非射门练习，也非多人竞技，而是单人表演者完成的一整套连续花式足球动作。
整体氛围应包括：街头感、城市感、电影感、运动感、酷炫且富有节奏感。
背景设定在夜间的小型城市街头球场或城市广场足球空间，包含涂鸦墙、铁丝网、远处的建筑灯光、路灯、略带反光的地面；环境应统一呈现出街头潮流风格。

[人物要求]
主角是一位年轻帅气的男性花式足球专家，体格轻盈协调且富有运动感，短黑发，神情自信，动作流畅且具有控制力。
整张图片的 16 个格子必须是同一个人；请勿在每个格子中更换人物。

[布局要求]
整张图片为 4x4 十六宫格，每个格子必须包含：
1. 大号步骤数字（1-16）
2. 中文动作标题
3. 2-4 个简短的中文动作要点
4. 清晰的人物全身动作
5. 足球轨迹箭头 / 脚法轨迹线
6. 足球必须清晰可见且位置合理，不得悬空或消失。

整体字体风格应偏向运动海报美学，清晰有力，适合社交媒体分享。图像应锐利、清晰且干净；人物和足球都应完整呈现，不得切掉脚部。

[动作序列]
请按以下顺序设计 16 个连续动作，每个动作各不相同，形成完整的花式表演流：

1. 街头开场姿势
动作：单脚踩球，身体舒展，自信站姿，如同表演开始。
要点：主导站姿 / 右脚踩球 / 展示自信
2. 脚底回拉
动作：用脚底将球向后滚动，增加轻微的横向移动。
要点：脚底触球 / 向后拉球 / 轻微转体
3. 跨步绕球
动作：双脚围绕足球进行跨步绕球。
要点：单脚绕球 / 跨步换脚 / 平滑节奏
4. 内外侧运球
动作：使用脚内侧和外侧连续触球，左右切换。
要点：内侧触球 / 外侧触球 / 快速切换
5. 鞋带挑球
动作：用鞋带部位将球向上挑起，使其离开地面。
要点：鞋带发力 / 挑起球 / 球在前方升起
6. 连续颠球
动作：用鞋带部位连续颠球，保持球在身体前方上下弹跳。
要点：交替颠球 / 保持节奏 / 稳定连贯
7. 膝盖控制
动作：利用膝盖或大腿控制球的弹跳。
要点：球击中膝盖 / 控制落点 / 保持平衡
8. 脚后跟挑球
动作：利用脚后跟从后方或侧后方将球挑起。
要点：脚后跟接触 / 向上挑球 / 身体前倾
9. 环绕地球 (ATW)
动作：单脚绕过空中的球。
要点：脚向外绕 / 完整一圈 / 稳定的落地控制
10. 交叉环绕地球
动作：交叉双腿完成绕球动作。
要点：前腿交叉 / 完整一圈 / 空中切换
11. 肩部停球
动作：将球停在肩部或颈部附近。
要点：球落在肩上 / 收紧肩部 / 稳定停球
12. 背部停球
动作：身体前倾，让球短暂留在背部。
要点：身体前倾 / 球在背上 / 保持平衡
13. 彩虹过人
动作：从身后将球挑过头顶，形成彩虹轨迹。
要点：脚后跟挑球 / 球过头顶 / 弧线轨迹
14. 原地旋转接球
动作：原地旋转并接住下落的球。
要点：原地旋转 / 接住下落球 / 轻柔落地控制
15. 脚尖停球
动作：利用脚尖或鞋面精准停球。
要点：脚尖抬起 / 精准停球 / 完美平衡
16. 街头谢幕
动作：再次单脚踩球，双臂张开，如同完整表演的结束。
要点：右脚踩球 / 张开双臂 / 帅气谢幕

[视觉风格]
- 整体高端运动海报风格
- 电影感夜景灯光
- 强烈的动感，带有少许潮流街头美学
```
