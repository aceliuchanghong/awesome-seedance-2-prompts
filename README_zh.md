## 🤔 什么是 Seedance 2.0？

**Seedance 2.0** 是**字节跳动**开发的视频生成模型，是业内首个支持**四模态同时输入**的模型——图片、视频、音频和文本。

**Key Features:**
- 🎥 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片动画化为动态视频
- 📹 **视频转视频** — 转换和扩展现有视频
- 🎵 **音频驱动** — 通过音频输入驱动视频生成
- 📐 **最高 1080p 分辨率**，4-15 秒时长
- 🔊 **自动配音配乐** — 自动旁白和背景音乐

一个粗犷的丛林主题公园入口。原始的木栅栏、高大的大门，以及一块巨大的悬挂木牌，上面写着“猫咪宇宙” (CAT UNIVERSE)。远处是郁郁葱葱的丛林。空气中弥漫着薄雾。氛围略显诡异。

一只穿着卡其色探险装备、脖子上挂着望远镜的橘猫坐在尘土飞扬的吉普车里。一只穿着粉色裙子的白猫坐在后座，显然对此毫无兴趣。

0–3 秒：
广角镜头。吉普车在门口停下。丛林中回荡着阵阵声响。

一名穿着迷彩服、戴着墨镜的鬣狗守卫走上前来，神情严肃且沉默。

他向驾驶座一侧倾身。

鬣狗守卫（语气平淡，充满威胁）：
“票。”

橘猫特写。自信的微笑……逐渐消失。

他开始疯狂地翻找。

3–6 秒：
车内混乱的特写镜头。

手套箱弹开——纸张、零食包装纸、空罐头飞得到处都是。

橘猫到处乱翻。恐慌感加剧。

橘猫（尴尬地笑）：
“好吧……说来话长……我绝对是带了……在精神层面上。”

切换到后座的白猫，盯着窗外，一脸无聊和烦躁。

白猫（冷淡地）：
“我就知道这次约会不合法。”

橘猫僵住了。看向守卫。

橘猫（尴尬）：
“那个……关于票的事……我没带。”

停顿。一片寂静。

6–9 秒：
突变。

橘猫猛踩油门。

引擎轰鸣。

吉普车猛地冲了出去——

撞碎了木门。木屑在慢动作中飞溅。

鬣狗守卫向后跳开，大喊道。

鬣狗守卫：
“嘿——那不是有效的——”

声音被混乱掩盖。

吉普车冲进丛林。

白猫尖叫着抓住座椅。

白猫：
“太棒了，现在我们成罪犯了——”

9–12 秒：
丛林深处。镜头跟随吉普车疾驰。

发光的漂浮气泡充斥着空气——每一个气泡都闪烁着不同的色彩。

每个气泡里：都有一只正在睡觉的绿色小霸王龙宝宝。

吉普车疾驰而过——

气压扰动了气泡。

砰——砰——砰——

气泡迅速破裂。

小霸王龙宝宝掉落，瞬间惊醒——双眼发光，露出獠牙。

它们尖叫着开始追逐吉普车。

快速混乱的追逐。

白猫叹了口气，完全受够了。

白猫（烦躁，面无表情）：
“看吧？这就是为什么我不能让你做决定。”

12–15 秒：
她伸手到座椅后——

随手掏出一个火箭发射器。

橘猫回头看了一眼，一脸担忧。

橘猫：
“……你约会带这个？”

白猫耸耸肩。

白猫：
“我喜欢多几个选择。”

她开火了。

嗖——

火箭在半空中分裂成多个燃烧的碎片。

每个碎片瞄准一只不同的霸王龙宝宝。

快速撞击——产生微小的爆炸。

霸王龙们哀鸣着，惊慌失措，四散奔逃。

除了引擎的嗡嗡声，一切归于平静。

最终镜头：
吉普车内。
