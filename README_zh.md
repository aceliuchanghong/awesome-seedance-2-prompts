## 🤔 什么是 Seedance 2.0？

**Seedance 2.0** 是**字节跳动**开发的视频生成模型，是业内首个支持**四模态同时输入**的模型——图片、视频、音频和文本。

**Key Features:**
- 🎥 **文本生成视频** — 根据文字描述生成视频
- 🖼️ **图片生成视频** — 将静态图片动画化为动态视频
- 📹 **视频转视频** — 转换和扩展现有视频
- 🎵 **音频驱动** — 通过音频输入驱动视频生成
- 📐 **最高 1080p 分辨率**，4-15 秒时长
- 🔊 **自动配音配乐** — 自动旁白和背景音乐

广角镜头。风声呼啸。
黑猫轻轻拍了拍老虎（老虎发出吱吱声）。

橘猫：
“是啊，人就是这么消失的。我去看看。”
黑猫点点头。

橘猫举起一根手指——
一把廉价手电筒突然出现在他手中。
橘猫（冷淡）：
“如果我没回来，替我报仇……或者至少装装样子。”
他乘着云朵滑进洞穴。
6–9 秒：
洞内。黑暗。
他拨开蛛网，一脸烦躁。
橘猫：
“太棒了。清洁预算为零。”
他发现深处有光——

灰猫（冷漠）：
“看。一个志愿者。”
橘猫落下，手中出现长棍。
橘猫：
“够了。怪物不会得到两次警告。”
战斗爆发：
他旋转、出击、翻滚——动作迅速、华丽。

灰猫们（嘲讽）：
“真可爱。”
“他为此训练过呢。”
橘猫感到挫败——瞄准其中一只——跃起准备终结一击——
12–14 秒：
两根手指突然在半空中挡住了长棍。
揭晓：黑猫出现在他身后。
黑猫（温柔）：
“我们不伤害生灵。”
瞬间转变——

黑猫保持沉默，神情安详。
尴尬的停顿。
切至黑屏，伴随一声尖锐的喜剧音效。
